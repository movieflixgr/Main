Πολιτική Απορρήτου - MovieFlix

Η εφαρμογή MovieFlix παρέχεται δωρεάν από τη Med1a Network και προορίζεται για προσωπική χρήση. Η παρούσα σελίδα χρησιμοποιείται για να ενημερώσει τους επισκέπτες σχετικά με τις πολιτικές μας για τη συλλογή, χρήση και αποκάλυψη Προσωπικών Στοιχείων, εφόσον κάποιος αποφασίσει να χρησιμοποιήσει την υπηρεσία μας.

Εάν επιλέξετε να χρησιμοποιήσετε την υπηρεσία μας, συμφωνείτε με τη συλλογή και χρήση πληροφοριών σύμφωνα με αυτήν την πολιτική. Τα Προσωπικά Στοιχεία που συλλέγουμε χρησιμοποιούνται για την παροχή και βελτίωση της υπηρεσίας μας. Δεν θα χρησιμοποιήσουμε ή θα μοιραστούμε τις πληροφορίες σας με οποιονδήποτε εκτός από ό,τι περιγράφεται σε αυτήν την Πολιτική Απορρήτου.

Συλλογή και Χρήση Πληροφοριών:

Για καλύτερη εμπειρία κατά τη χρήση της υπηρεσίας μας, ενδέχεται να σας ζητηθεί να παρέχετε ορισμένες προσωπικά αναγνωρίσιμες πληροφορίες, συμπεριλαμβανομένου, αλλά όχι περιοριστικά, του αναγνωριστικού διαφήμισης Android. Οι πληροφορίες που ζητούμε θα διατηρούνται στη συσκευή σας και δεν συλλέγονται από εμάς με κανέναν τρόπο.

Υπηρεσίες Τρίτων:

Η εφαρμογή χρησιμοποιεί υπηρεσίες τρίτων που ενδέχεται να συλλέγουν πληροφορίες που χρησιμοποιούνται για τον αναγνωρισμό σας. Ελέγξτε την πολιτική απορρήτου των υπηρεσιών που χρησιμοποιούμε παρακάτω:

  • Google Play Services
  • AdMob
  • Firebase Analytics
  • Firebase Crashlytics

Δεδομένα Καταγραφής (Log Data):

Σας ενημερώνουμε ότι κατά τη χρήση της υπηρεσίας μας, σε περίπτωση σφάλματος στην εφαρμογή, συλλέγουμε δεδομένα και πληροφορίες (μέσω προϊόντων τρίτων) από το τηλέφωνό σας, γνωστά ως Δεδομένα Καταγραφής. Αυτά τα δεδομένα μπορεί να περιλαμβάνουν πληροφορίες, όπως η διεύθυνση Internet Protocol ("IP") της συσκευής σας, το όνομα της συσκευής, η έκδοση του λειτουργικού συστήματος, η διαμόρφωση της εφαρμογής κατά τη χρήση της υπηρεσίας, η ώρα και η ημερομηνία χρήσης της υπηρεσίας και άλλα στατιστικά στοιχεία.

Cookies:

Τα "cookies" είναι αρχεία με μικρό όγκο δεδομένων που συνήθως χρησιμοποιούνται ως ανώνυμοι μοναδικοί αναγνωριστικοί. Αν και η υπηρεσία μας δεν χρησιμοποιεί αυτά τα "cookies" ρητά, η εφαρμογή ενδέχεται να χρησιμοποιεί κώδικα και βιβλιοθήκες τρίτων που χρησιμοποιούν "cookies" για τη συλλογή πληροφοριών και τη βελτίωση των υπηρεσιών τους. Έχετε τη δυνατότητα να αποδεχτείτε ή να αρνηθείτε αυτά τα "cookies" και να ενημερώνεστε όταν ένα "cookie" στέλνεται στη συσκευή σας. Η αρνητική απάντηση στα "cookies" ενδέχεται να σας αποκλείσει από τη χρήση ορισμένων τμημάτων της υπηρεσίας μας.

Παροχείς Υπηρεσιών:

Ενδέχεται να χρησιμοποιούμε εταιρείες και άτομα τρίτων για τη διευκόλυνση, την παροχή και την ανάλυση της υπηρεσίας μας. Θέλουμε να ενημερώσουμε τους χρήστες ότι αυτοί οι τρίτοι έχουν πρόσβαση στα Προσωπικά Σας Στοιχεία για να εκτελέσουν τις εργασίες που τους έχουν ανατεθεί εκ μέρους μας. Ωστόσο, έχουν την υποχρέωση να μην αποκαλύπτουν ή να χρησιμοποιούν τις πληροφορίες για οποιοδήποτε άλλο σκοπό.

Ασφάλεια:

Εκτιμούμε την εμπιστοσύνη που μας παρέχετε παρέχοντάς μας τα Προσωπικά Σας Στοιχεία. Εργαζόμαστε για τη χρήση εμπορικά αποδεκτών μέσων προστασίας. Ωστόσο, να σημειώσετε ότι καμία μέθοδος μετάδοσης μέσω Διαδικτύου ή μέθοδος ηλεκτρονικής αποθήκευσης δεδομένων δεν είναι 100% ασφαλής και αξιόπιστη, και δεν μπορούμε να εγγυηθούμε την απόλυτη ασφάλεια.

Συνδέσεις προς Άλλες Ιστοσελίδες:

Η υπηρεσία μας μπορεί να περιέχει συνδέσμους προς άλλες ιστοσελίδες. Εάν κάνετε κλικ σε έναν σύνδεσμο τρίτου μέρους, θα μεταφερθείτε στην ιστοσελίδα αυτού του τρίτου μέρους. Σημειώστε ότι αυτές οι εξωτερικές ιστοσελίδες δεν είναι υπό τον έλεγχό μας. Δεν έχουμε καμία ευθύνη για το περιεχόμενο, τις πολιτικές απορρήτου ή τις πρακτικές οποιωνδήποτε ιστοσελίδων ή υπηρεσιών τρίτων. Σας προτείνουμε να εξετάσετε τις πολιτικές απορρήτου των εκάστοτε ιστοσελίδων που επισκέπτεστε.

Αλλαγές στην Πολιτική Απορρήτου:

Μπορεί να ενημερώνουμε την Πολιτική Απορρήτου μας από καιρό σε καιρό. Σας προτείνουμε να ελέγχετε περιοδικά αυτήν τη σελίδα για τυχόν αλλαγές. Θα σας ενημερώσουμε για οποιεσδήποτε αλλαγές αν δημοσιεύσουμε τη νέα Πολιτική Απορρήτου σε αυτή τη σελίδα. Οι αλλαγές στην Πολιτική Απορρήτου είναι αποτελεσματικές μόλις δημοσιευτούν σε αυτήν τη σελίδα.
